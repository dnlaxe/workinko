# Plan

## Goal

Job board.
No accounts.
Posters post and manage listings via magic links.
Job seekers browse and contact employers through a hidden email relay.
Admin approves everything before it goes live.

## Stack

Express, PostgreSQL, Drizzle ORM, express-handlebars, Pino, Tailwind, TypeScript

## V1

- [ ] Foundation — Express, HBS, DB, env validation
- [ ] Submission form — multi-step, basket, saves to DB on checkout
- [ ] Email — magic links, contact relay
- [ ] Admin — approve/reject listings and contacts
- [ ] Listings — browse, detail, contact form, stats
- [ ] Magic link management — edit, delete, renew, expiry
- [ ] Deploy
